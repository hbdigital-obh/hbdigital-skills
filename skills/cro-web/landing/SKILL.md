
## Règles de construction

**Ce qui est obligatoire :**
- CTA visible sans scroller (above the fold)
- H1 qui répond à "qu'est-ce que j'y gagne ?" pas à "qui êtes-vous ?"
- Mobile-first (tester sur 375px)
- Formulaire < 4 champs si l'objectif est un lead de premier niveau
- Temps de chargement < 3 secondes (pas de vidéo en autoplay lourde)
- Une seule navigation ou aucune (pas de menu qui fait fuir)

**Ce qui est interdit :**
- Carrousel en hero (les photos défilantes tuent la conversion)
- Plusieurs CTA différents (un seul objectif = un seul CTA)
- Texte sur fond image illisible
- Prix barrés permanents sur une marque premium (SEYDI Paris)
- Pop-up qui s'ouvre au chargement (avant que l'utilisateur lise quoi que ce soit)

## Copywriting par section

### H1 — La formule qui marche
```
Option A — Résultat : "Obtenez [résultat] en [délai] sans [obstacle principal]"
Option B — Question : "Tu veux [désir] mais [frein] ?"
Option C — Déclaration : "[Résultat] pour [cible précise]"
```

### CTA — Textes qui convertissent
```
❌ "Envoyer" / "Valider" / "Cliquer ici"
✅ "Recevoir mon devis gratuit"
✅ "Calculer mon économie d'impôts"
✅ "Découvrir la collection en édition limitée"
✅ "Réserver ma place (gratuit)"
```

### Preuve sociale — Formats efficaces
```
Chiffres : "+2 300 clients accompagnés depuis 2019"
Presse : logo Marie France / Biba (SEYDI) ou logo média B2B
Témoignage : "[Résultat précis] en [délai]. — [Prénom], [poste/ville]"
Certification : labels, partenariats, organismes
```

## Templates par secteur

### Lead gen défiscalisation
- H1 : "Économisez jusqu'à [X€] d'impôts dès cette année"
- CTA : "Calculer mon économie"
- Formulaire : Prénom / Email / Téléphone / Tranche d'imposition

### Lead gen assurance / énergie
- H1 : "Comparez les meilleures offres [assurance/énergie] en 2 minutes"
- CTA : "Voir les offres disponibles"
- Formulaire : Code postal / Email / Téléphone

### E-commerce SEYDI Paris
- H1 : "Mode éthique, wax africain, fabriqué en France"
- Sous-titre : "Collections capsules en édition limitée. Chaque pièce a une histoire."
- CTA : "Découvrir la collection"
- Pas de formulaire — lien vers la boutique Shopify

## Format de réponse

Pour chaque demande de landing :
1. **Brief** : objectif, cible, offre, trafic source
2. **Structure** : wireframe textuel section par section
3. **Copie complète** : H1, sous-titre, corps, CTA, formulaire
4. **Specs** : recommandations visuelles (couleurs, images, mise en page)
5. **Code HTML** : si demandé, structure de base responsive
