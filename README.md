# hbdigital-skills
Custom Claude skills for HB Digital.

## Purpose
This repository contains reusable `SKILL.md` files that extend Claude with task-specific workflows, prompts, and guardrails.

## Repository layout
- `README.md`: project overview and contribution workflow.
- `CLAUDE.md`: guidance for Claude when editing this repository.
- `*/SKILL.md`: one skill per directory (recommended structure).

## Getting started
1. Clone the repository.
2. Create a folder for a skill.
3. Add a `SKILL.md` file with clear trigger conditions and instructions.
4. Commit changes and push to `main` (or via pull request if you prefer review).

## Contribution guidelines
- Keep each skill focused on one job.
- Define explicit triggers (when the skill should be used).
- Add concrete, testable instructions.
- Avoid ambiguous wording and duplicated scope across skills.
