# Content Production — HB Digital Performance

## Contexte production HB Digital

La production de contenu chez HB Digital couvre :
- Articles SEO pour sites propriétaires (astuces, comparatifs, défisc)
- Contenus marketing pour clients (annonceurs, affiliés)
- Contenus SEYDI Paris (mode éthique)
- Kits de communication et copy pour campagnes

**Ressources de production typiques :**
- Rédacteurs freelance (brief → livraison → révision)
- Claude (production directe ou révision)
- Équipe interne (validation finale + publication)

### 1.2 Délais standards par format

| Format | Longueur | Délai standard |
|---|---|---|
| Article court | 500-800 mots | 2 jours ouvrés |
| Article moyen | 800-1 500 mots | 3-4 jours ouvrés |
| Article long (pilier) | 1 500-3 000 mots | 5-7 jours ouvrés |
| Fiche produit (série) | 150-300 mots × N | 1 jour pour 5 fiches |
| Email / newsletter | 200-400 mots | 1-2 jours ouvrés |
| Script vidéo | 3-5 min | 2-3 jours ouvrés |

## ÉTAPE 2 — Workflow de production

### 2.1 Processus standard (5 étapes)

```
ÉTAPE 1 — BRIEF (J0)
├── Hady valide le sujet et la stratégie
├── Brief rédacteur généré (ce skill)
└── Brief envoyé + deadline confirmée

ÉTAPE 2 — PRODUCTION (J1 → J+X)
├── Rédacteur ou Claude produit le contenu
└── Livraison dans Google Doc partagé

ÉTAPE 3 — RÉVISION TECHNIQUE (J+X)
├── Vérification SEO (mots-clés, structure Hn, longueur)
├── Vérification factuelle
└── Vérification ton éditorial

ÉTAPE 4 — VALIDATION (J+X+1)
├── Lecture finale Hady (ou délégation)
├── Corrections mineures si nécessaire
└── Validation GO/NO-GO

ÉTAPE 5 — PUBLICATION (J+X+1 ou planifié)
├── Mise en ligne WordPress / plateforme
├── Ajout métas + images + liens internes
└── Mise à jour calendrier éditorial
```

### 2.2 Statuts de suivi (Notion / Trello)

| Statut | Signification |
|---|---|
| 📋 Brief envoyé | Brief transmis au rédacteur |
| ✍️ En production | Rédacteur travaille dessus |
| 🔍 En révision | Contenu livré, en cours de relecture |
| ✅ Validé | Approuvé, prêt à publier |
| 🚀 Publié | En ligne + URL enregistrée |
| 🔄 Révision demandée | Retour au rédacteur pour corrections |

## ÉTAPE 3 — Grille de révision qualité

### 3.1 Check SEO

- [ ] Mot-clé principal dans H1 ?
- [ ] Mot-clé principal dans les 100 premiers mots ?
- [ ] Au moins 1 H2 avec mot-clé secondaire ?
- [ ] Méta title < 60 caractères ?
- [ ] Méta description < 155 caractères + mot-clé + CTA implicite ?
- [ ] Longueur conforme au brief ?
- [ ] Au moins 1 lien interne ?

### 3.2 Check éditorial

- [ ] Intro accroche en moins de 3 phrases ?
- [ ] Structure Hn logique et progressive ?
- [ ] Pas de paragraphe > 5 lignes ?
- [ ] Conclusion avec CTA clair ?
- [ ] Ton conforme aux guidelines du projet ?
- [ ] Pas de fautes d'orthographe / syntaxe grossières ?

### 3.3 Check conformité

- [ ] Pas de promesses illégales (défiscalisation, assurance) ?
- [ ] Sources citées si chiffres avancés ?
- [ ] Mentions légales requises si applicable ?

## ÉTAPE 4 — Gestion des freelances

### 4.1 Onboarding rédacteur externe

```
Kit à envoyer à un nouveau rédacteur :
1. Guide éditorial du projet (1 page)
2. Exemple d'article validé + annoté
3. Template Google Doc avec structure pré-remplie
4. Processus de livraison et de révision
5. Grille tarifaire et conditions de paiement
```

### 4.2 Évaluation de la qualité

Scoring rapide après chaque livraison (note sur 5 par critère) :

| Critère | Note /5 |
|---|---|
| Respect du brief (structure, longueur, mots-clés) | |
| Qualité rédactionnelle (fluidité, clarté) | |
| Ton éditorial respecté | |
| Délai respecté | |
| Nombre de révisions nécessaires | |

**Seuil minimal acceptable : 3/5 sur tous les critères.**
En dessous : feedback constructif + nouvelle chance.
Deux livraisons < 3/5 : chercher un nouveau rédacteur.

## RÈGLES ABSOLUES

1. Chaque contenu a un brief écrit avant de démarrer la production
2. Template Google Doc fourni au rédacteur — pas d'article dans Word ou email
3. Révisions maximum : 2 rounds — au-delà, discuter du brief
4. Jamais publier sans passer la grille de révision qualité
5. Finir par 1 action précise pour avancer la production
