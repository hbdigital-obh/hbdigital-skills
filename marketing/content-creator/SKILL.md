## ÉTAPE 0 — Self-healing

| Situation | Comportement |
|---|---|
| Canal non précisé | Demander 1 seule question ou produire pour Blog + LinkedIn |
| Longueur non précisée | Appliquer le standard canal (voir tableau ci-dessus) |
| Sujet vague | Déduire depuis le projet + proposer 3 angles |
| Mots-clés SEO non fournis | Suggérer les 3 mots-clés principaux évidents |
| Ton non précisé | Appliquer le ton du projet détecté |

## ÉTAPE 2 — Structures par format

### 2.1 Article de blog (SEO)

```
STRUCTURE STANDARD :

# [Titre H1 — mot-clé principal — < 60 caractères]
*[Méta description — 150 caractères — bénéfice + mot-clé]*

**Introduction (100-150 mots)**
- Accrocher avec un chiffre ou une question
- Annoncer le problème traité
- Promettre ce que le lecteur va apprendre

## [H2 — Sous-thème principal 1]
[Corps 200-300 mots]

## [H2 — Sous-thème principal 2]
[Corps 200-300 mots]

## [H2 — Sous-thème principal 3]
[Corps 200-300 mots]

## [H2 — FAQ ou "En résumé"]
[3-5 questions/réponses courtes]

**Conclusion (80-100 mots)**
- Récapituler en 2 phrases
- CTA clair (lien interne ou formulaire)
```

**Règles SEO intégrées :**
- Mot-clé principal dans : title, H1, premier paragraphe, 1 H2, méta description
- Mots-clés secondaires naturellement intégrés
- 1 lien interne minimum vers une page du site
- Alt text pour chaque image suggérée

### 2.2 Post LinkedIn

```
STRUCTURE HAUTE PERFORMANCE :

[Ligne 1 : Hook — arrêter le scroll]
(Question provocante / affirmation contre-intuitive / chiffre inattendu)

[Ligne 2 : vide — forcer le "Voir plus"]

[Corps : 3-5 paragraphes courts]
- 1 idée par paragraphe
- Phrases courtes (< 20 mots)
- Alternance : constat → exemple → insight

[Conclusion : 1-2 phrases — enseignement ou appel à réaction]

[CTA discret : question pour les commentaires]

#hashtag1 #hashtag2 #hashtag3 (3 max, en fin de post)
```

**Types de posts LinkedIn pour HB Digital :**
- Chiffre d'affaires / cas client (autorité)
- Erreur à éviter en lead gen (valeur + opinion)
- Thread "comment on fait X chez HB Digital" (pédagogique)
- Actualité secteur commentée (expertise)

### 2.3 Caption Instagram (SEYDI Paris)

```
STRUCTURE :

[1ère phrase : accroche visuelle ou émotionnelle]
(Ce que le produit/moment fait ressentir)

[2ème bloc : histoire courte ou origine]
(Tissu, artisan, savoir-faire, valeur portée)

[3ème bloc : invitation]
(Inciter à swiper, commenter, visiter la boutique)

[Hashtags : 8-15, regroupés en fin]
#seydiparis #modeethnique #waxfabric #madeinfrance [sectoriels]
```

### 2.4 Email newsletter

```
STRUCTURE :

[Objet : 5-8 mots — curiosité ou bénéfice direct]
[Preheader : 10-15 mots — compléter l'objet]

---

[Salutation personnalisée ou directe]

[Intro : 2-3 phrases — contexte ou actu]

[Corps principal : 1 idée, 1 histoire, 1 outil ou 1 insight]
(150-250 mots max)

[CTA principal : bouton ou lien texte — 1 seul]

[PS optionnel : 1 info bonus courte]

---
[Footer légal + désabonnement]
```

### 2.5 Script vidéo / YouTube

```
STRUCTURE (pour vidéo 3-5 min) :

[Hook 0-10 sec] : Promesse ou question directe
[Intro 10-30 sec] : Contexte rapide + ce qu'on va voir
[Développement 3 parties — 80% du temps]
  - Partie 1 : [X minutes]
  - Partie 2 : [X minutes]
  - Partie 3 : [X minutes]
[CTA final 15-30 sec] : Subscribe / lien en description / commentaire

RÈGLES :
- 1 idée par 30-60 secondes
- Transitions claires entre les parties
- Répéter le bénéfice principal 2-3 fois
- Finir sur une question pour les commentaires
```

## RÈGLES ABSOLUES

1. Toujours adapter la longueur au canal — jamais d'article trop court, jamais de post trop long
2. Contenu prêt-à-copier-coller — pas de placeholders en fin de livraison
3. SEO : au moins 1 mot-clé dans le titre si contenu blog
4. LinkedIn : hook fort sur la ligne 1 — sinon le post ne sera pas lu
5. Finir par 1 recommandation sur le contenu suivant dans la série
