## ÉTAPE 1 — Diagnostic du besoin copy

### 1.1 Identifier le format

| Format | Longueur cible | Enjeu principal |
|---|---|---|
| Headline LP | 6-12 mots | Accrocher en 3 secondes |
| Sous-titre LP | 1-2 phrases | Confirmer le bénéfice |
| Body LP | 150-400 mots | Convaincre + supprimer les objections |
| CTA | 2-5 mots | Déclencher l'action |
| Objet email | 4-8 mots | Ouvrir |
| Preheader email | 8-15 mots | Compléter l'objet |
| Body email cold | 60-100 mots | Réponse / RDV |
| Ad Facebook | Headline 5 mots + body 40 mots | Arrêt du scroll |
| Ad Google | Titre 30 car × 3 + desc 90 car × 2 | Clic qualifié |

### 1.2 Identifier l'angle

**Les 7 angles universels :**
1. **Peur / Perte** : "Ce que vous perdez chaque mois sans…"
2. **Gain rapide** : "Obtenez X en Y jours"
3. **Curiosité** : "La méthode que les X n'utilisent pas encore"
4. **Preuve sociale** : "Comment X a fait Y avec Z"
5. **Autorité** : "Selon les experts…" / Données chiffrées
6. **Nouveauté** : "La nouvelle façon de…"
7. **Simplicité** : "Arrêtez de complexifier. Voici comment…"

## ÉTAPE 2 — Formules de copy

### 2.1 Headlines

**Formule AIDA :**
```
A (Attention) : Chiffre ou question provocante
I (Intérêt) : Bénéfice inattendu
D (Désir) : Transformation souhaitée
A (Action) : Invitation directe

Exemple :
"3 annonceurs sur 4 perdent de l'argent sur leurs leads.
Pas ceux qui travaillent avec nous."
```

**Formule Problem-Agitate-Solve (PAS) :**
```
P : Nommer le problème précis
A : Amplifier la douleur (coût, frustration, conséquences)
S : Présenter la solution

Exemple :
"Vous payez des leads que vos commerciaux n'arrivent pas à joindre.
Chaque lead non converti, c'est de l'argent qui part.
Nos leads sont vérifiés, opt-in, et livrés en moins de 24h."
```

**Formule Before-After-Bridge (BAB) :**
```
Avant : situation actuelle douloureuse
Après : situation idéale souhaitée
Bridge : votre offre comme pont entre les deux
```

### 2.2 CTAs qui convertissent

**Éviter :** "Cliquez ici" / "En savoir plus" / "Envoyer"

**Utiliser :**
```
Bénéfice immédiat :
→ "Recevoir mes leads défiscalisation"
→ "Obtenir mon audit gratuit"
→ "Démarrer ma campagne"

Urgence :
→ "Réserver ma place (3 restantes)"
→ "Accéder maintenant — offre limitée"

Réduction friction :
→ "Essayer gratuitement — sans engagement"
→ "Voir un exemple de lead"
```

### 2.3 Objections à éliminer

**Les 5 objections universelles à traiter dans tout copy de conversion :**

| Objection | Réponse copy |
|---|---|
| "C'est trop cher" | Montrer le ROI / coût de l'inaction |
| "Ça ne marchera pas pour moi" | Témoignage similaire + données |
| "Je n'ai pas le temps" | Montrer la facilité / délai précis |
| "Je ne vous fais pas confiance" | Preuves sociales + garanties |
| "Je dois y réfléchir" | Urgence + conséquences de l'attente |

### 2.4 Copy ads performance (Meta / Google)

**Structure ad Meta :**
```
[Hook — 1 phrase] : accrocher dans le fil d'actualité
[Problème] : 1-2 phrases — s'identifier
[Solution] : 1-2 phrases — l'offre
[Preuve] : 1 chiffre ou témoignage court
[CTA] : 1 phrase d'action

Longueur totale : < 150 mots
```

**Structure annonce Google :**
```
Titre 1 (30 car) : Mot-clé principal + bénéfice
Titre 2 (30 car) : Différenciateur ou offre
Titre 3 (30 car) : CTA direct
Description 1 (90 car) : Bénéfices + preuve
Description 2 (90 car) : Réduction objection + CTA secondaire
```

## ÉTAPE 3 — Copy par secteur HB Digital

### Défiscalisation (LPs, emails)

```
Angles qui fonctionnent :
- Gain fiscal chiffré ("Réduisez vos impôts de X €")
- Urgence législative ("Avant le 31 décembre")
- Simplicité ("Un conseiller s'occupe de tout")
- Preuve ("X projets accompagnés en 2024")

À éviter :
- Promesses irréalistes
- Termes trop techniques sans explication
- Ton alarmiste excessif
```

### Assurance / Énergie

```
Angles qui fonctionnent :
- Économies chiffrées ("Économisez jusqu'à X €/an")
- Comparaison rapide ("Meilleure offre en 2 minutes")
- Pas d'engagement ("Résiliable à tout moment")

À éviter :
- Jargon assuranciel (franchise, cotisation…) sans explication
- Promesses sur les garanties sans nuance
```

### SEYDI Paris (mode éthique)

```
Angles qui fonctionnent :
- Origine et fabrication ("Tissus wax + fait en France")
- Valeurs et engagement ("Mode qui a du sens")
- Sensorialité ("Le toucher du coton, l'éclat des couleurs")
- Communauté ("Rejoindre les femmes qui…")

À éviter :
- Greenwashing vague
- "Eco-responsable" sans preuve
- Surenchère sur les prix
```

## ÉTAPE 4 — Checklist qualité copy

Avant livraison, vérifier :

- [ ] Headline dit "et alors pour moi ?" en 3 secondes ?
- [ ] Bénéfice avant caractéristique ?
- [ ] Au moins 1 chiffre ou preuve concrète ?
- [ ] Une seule action demandée (pas deux CTA concurrents) ?
- [ ] Objections principales traitées ?
- [ ] Lecture fluide à voix haute ?
- [ ] Pas de jargon interne (on dit "vous obtenez", pas "nos services vous permettent de") ?

## RÈGLES ABSOLUES

1. Bénéfice avant caractéristique — toujours
2. 1 CTA unique par pièce de copy — pas de bifurcation
3. Concret > abstrait : chiffres, faits, exemples > affirmations creuses
4. Toujours produire au moins 2 variantes (angles différents) pour tester
5. Finir par une recommandation de test (A/B ou multivarié)
