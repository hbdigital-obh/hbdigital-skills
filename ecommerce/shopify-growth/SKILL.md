# SKILL — Shopify Growth (shopify-growth)

## Contexte SEYDI Paris

**Marque :** SEYDI Paris — mode féminine éthique
**Positionnement :** tissu wax africain + savoir-faire made in France, pièces uniques ou petites séries
**Cible :** femmes 25–45 ans, CSP+, sensibles à l'origine et à l'impact, afro-diaspora et alliées
**Canal principal :** Shopify + Meta Ads + Instagram organique
**Saisonnalité :** pics Noël / fêtes africaines (Tabaski, Noël, Saint-Valentin) / Été

**KPIs cibles SEYDI :**
- Taux de conversion : ≥ 2,5%
- AOV (Average Order Value) : ≥ 80 €
- Taux d'abandon panier : < 65%
- Email open rate (Klaviyo) : ≥ 35%
- Repeat purchase rate : ≥ 25% à 12 mois
- ROAS Meta : ≥ 3× en hors promo

## ÉTAPE 1 — Audit Shopify express

### Checklist 15 points (score /15)

```
PERFORMANCE TECHNIQUE
□ 1. Vitesse mobile PageSpeed ≥ 70 (GTmetrix / Lighthouse)
□ 2. Core Web Vitals : LCP < 2.5s, CLS < 0.1, INP < 200ms
□ 3. Images WebP, lazy load, pas d'images > 200kb

CONVERSION
□ 4. Fiche produit : photos HD multiple angles + vidéo courte
□ 5. Description : bénéfices d'abord, histoire tissu/fabrication
□ 6. Avis clients visibles above fold (app : Judge.me ou Yotpo)
□ 7. Taille guide clair et visible (mode : critique pour le taux de retour)
□ 8. Sticky Add to Cart sur mobile
□ 9. Pop-up collecte email (timing ≥ 8s, offre : -10% ou contenu exclusif)

PANIER & CHECKOUT
□ 10. Upsell en panier (app : Frequently Bought Together ou CartHook)
□ 11. Livraison gratuite progressive visible (ex. "Plus que X€ pour la livraison offerte")
□ 12. Options paiement : CB + PayPal + Alma/Scalapay (paiement en 3×)

EMAIL & RETENTION
□ 13. Séquence abandon panier active (min. 3 emails)
□ 14. Séquence post-achat active (min. 2 emails)
□ 15. Segment clients VIP/répétés identifié et adressé
```

**Score :**
- 13–15 ✅ Boutique bien configurée → optimiser les ads
- 8–12 🟡 Bases à consolider → prioriser les quick wins
- < 8 🔴 Refonte partielle nécessaire avant scaling

## ÉTAPE 4 — Séquences email automatisées

### Séquence Abandon Panier (3 emails)

```
Email 1 — J+1h (taux ouverture cible : 45%)
Objet : "Tu as oublié quelque chose 👀"
Corps : Photo produit + 1 ligne storytelling + CTA "Finaliser ma commande"
Ton : chaleureux, pas insistant

Email 2 — J+24h (taux ouverture cible : 30%)
Objet : "La pièce que tu aimes — l'histoire derrière"
Corps : Story du tissu (2-3 lignes) + avis client UGC + CTA
Pas de réduction à ce stade

Email 3 — J+72h (taux ouverture cible : 25%)
Objet : "Dernière chance + -10% rien que pour toi"
Corps : Urgence douce (stock limité si vrai) + code promo 10%
CTA : "Profiter de -10%"
```

### Séquence Post-Achat (4 emails)

```
Email 1 — Confirmation achat (J0, automatique Shopify)
  → Récap commande + timeline livraison

Email 2 — J+3 (avant livraison)
Objet : "Ta commande est en route ✨ + un cadeau pour toi"
Corps : Tracking + guide entretien tissu wax + lien Instagram communauté

Email 3 — J+10 (post-livraison estimée)
Objet : "Comment tu la portes ?"
Corps : Demande d'avis (Judge.me) + UGC encouragement (#seydiparis)

Email 4 — J+30 (fidélisation)
Objet : "Elles t'attendent aussi 🌿"
Corps : 3 nouvelles pièces complémentaires + offre fidélité (points Smile.io)
```

## ÉTAPE 5 — Stratégie Meta Ads (structure campagnes)

### Architecture campagnes SEYDI

```
NOTORIÉTÉ (Budget : 20% du total)
  → Vidéo 15s — story wax / fabrication
  → Audience : Lookalike 2-5% des acheteurs

CONSIDÉRATION (Budget : 30%)
  → Carrousel top 5 produits / collection
  → Audience : Visiteurs site 30j non acheteurs

CONVERSION (Budget : 40%)
  → DPA (Dynamic Product Ads) — catalogue Shopify
  → Remarketing panier abandonné 7j
  → Audience : Acheteurs Lookalike 1%

RÉTENTION (Budget : 10%)
  → Offre VIP / nouveautés / collection limitée
  → Audience : Acheteurs existants (Customer List)
```

### Creative brief ads SEYDI

```
Format performants :
- Reels 9:16, 15-30s : making-of, essayage, lifestyle
- Carrousel : 3-5 images produit + 1 image "histoire"
- Image statique : fond neutre + produit + tagline

Angles créatifs à tester :
A. "Made in France + wax africain" (différenciation)
B. "Porter son histoire" (identité / diaspora)
C. "Pièce unique, jamais vue ailleurs" (exclusivité)
D. UGC client réel (preuve sociale)
E. Behind the scenes atelier Paris (authenticité)
```

## ÉTAPE 6 — Optimisation du panier moyen (AOV)

### Tactiques immédiates

```
1. Seuil livraison gratuite = AOV actuel + 15-20%
   → Ex. si AOV 65€ → seuil à 79€

2. Bundle "Look complet"
   → Pièce principale + accessoire wax assorti
   → -5% sur le bundle (margin OK si ≥ 3×)

3. Paiement fractionné Alma visible dès la fiche produit
   → Message : "Dès 26€/mois en 3×"

4. Gift wrapping au checkout (3-5€)
   → Particulièrement pertinent cadeaux Noël / Tabaski

5. Carte cadeau Shopify (dénominations 30/50/100€)
   → Activer si pas encore fait
```

## RÈGLES ABSOLUES

1. Toujours respecter le positionnement premium de SEYDI — jamais de ton "discount agressif"
2. L'histoire du tissu et du savoir-faire est un actif différenciant — l'intégrer partout
3. Mobile-first : 80%+ du trafic SEYDI est mobile
4. Réductions max -15% (hors Black Friday -20%) — protéger la perception de valeur
5. UGC (User Generated Content) est le créatif le plus performant pour la mode éthique
6. Ne jamais promettre un ROAS sans connaître les marges réelles
7. Alma / paiement fractionné = levier AOV sous-utilisé à activer en priorité
